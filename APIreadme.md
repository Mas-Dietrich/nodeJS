## HTTP End Points:

### 1. GET TODOS

- **Endpoint**: `/todos/`

### 2. POST TODOS

- **Endpoint**: `/todos`

### 3. PUT TODO (Update)

- **Endpoint**: `/todos/:id`

### 4. DELETE TODO

- **Endpoint**: `/todos/:id`

### GET ALL TODOS for a CATEGORY

- **Endpoint**: `/todo/category/:category`

### GET CATEGORIES

- **Endpoint**: `/categories`


### POST CATEGORIES
- **Endpoint**: `/categories`

### PUT CATEGORIES (Update)

- **Endpoint**: `/categories/:category`

### DELETE CATEGORIES

- **Endpoint**: `/categories/:category`
